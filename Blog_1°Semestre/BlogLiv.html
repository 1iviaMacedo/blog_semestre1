Expo

Foi utilizado para dar início ao nosso projeto. Expo é uma ferramenta e uma plataforma para criar aplicações móveis usando React Native. Ela simplifica o processo de desenvolvimento ao fornecer uma série de ferramentas e bibliotecas prontas para uso, permitindo que os desenvolvedores se concentrem na criação de funcionalidades.

SQL

A gente usou para aumentar o aprendizado em criação de tabelas e o aprimoramento em filtragem de informações com base em dados específicos. SQL (Structured Query Language) é uma linguagem de programação utilizada para gerenciar e manipular bancos de dados relacionais. Com SQL, é possível realizar consultas, inserir, atualizar e excluir dados, além de criar e modificar a estrutura dos bancos de dados.

Arduino

Nosso aprendizado foi voltado a sensores, ao led e a montagem de um sistema básico de semáforo e a displays numéricos. Arduino é uma plataforma de prototipagem eletrônica open-source baseada em hardware e software fáceis de usar. É amplamente utilizado para criar projetos interativos, desde simples sensores até sistemas complexos de automação.

Node-RED

Foi utilizado para criar funções a partir de blocos fornecidos pelo próprio sistema do node red onde foi possivelvfazer com que essas funções funcionassem no arduino físico através de um complemento que ligou o app do arduino com o sistema do node. Node-RED é uma ferramenta de programação baseada em fluxo para conectar dispositivos de hardware, APIs e serviços online de novas maneiras. Com uma interface gráfica intuitiva, permite que desenvolvedores criem aplicações de integração com facilidade.

Ubuntu Server

Cria uma máquina virtual mais voltada aos códigos, foi por onde utilizamos o sistema do expo para a montagem do nosso app. Ubuntu Server é uma distribuição do sistema operacional Linux voltada para servidores. É conhecida pela sua estabilidade, segurança e grande comunidade de suporte, sendo amplamente utilizada em ambientes corporativos e servidores web.

Prompt

Prompt, ou linha de comando, é uma interface de texto onde os usuários podem digitar comandos para o sistema operacional executar. É uma ferramenta poderosa para administrar sistemas, automatizar tarefas e executar scripts. Foi interligado ao ubuntu server para utilizarmos a colagem de elementos no código mais facilmente porque é um prompt externo próprio do windows.

VisualG

VisualG é uma ferramenta educacional usada para ensinar lógica de programação. Com uma interface simples, permite criar algoritmos visuais e textuais, facilitando o aprendizado de conceitos básicos de programação para iniciantes. Ele foi importante para o aprimoramento do nosso conhecimento de JavaScript.

React Native

React Native é um framework para criar aplicativos móveis nativos usando JavaScript e React. Ele permite desenvolver para iOS e Android com um único código, garantindo alta performance e usabilidade. Assim como o html possui criação de classes e id (estilização style do html), a única diferença é o nome dado as tags.

Widgets

Widgets são componentes de interface que exibem informações e permitem interações rápidas diretamente na tela inicial de dispositivos, como smartphones e desktops. Eles facilitam o acesso a dados e funcionalidades sem abrir aplicativos completos.
